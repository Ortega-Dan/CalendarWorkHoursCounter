Calendar Hours Counter

  • Alt + K: Visible Week Report (Week-so-far if current week) (without optional filter [faster for 'In Progress' checks]).
  • Alt + I: Visible Week Report (Week-so-far if current week) (with optional filter for specific days query according to presented instructions in the interface).
  • Alt + O: Current Day 'Only' Report (with estimated finishing time).
  • Add Shift key to the previous options to count all visible events instead of only those that have actually passed before the current time (evaluates for entire week [not just week so far] if passed for the week options).

Works on Google Calendar for personal and enterprise versions, with English (US) language, and United States country settings, and counts events for the account main calendar, and not the ones from external or additional calendars.

It works in Week view of the calendar
(Switch to week view by hitting "w" key [Which is google-calendar's shortcut]).

To ignore events write cal.ignore (or c.ig) somewhere in the event title, separated from the rest of the text by some space or any other non-text character (any character distinct to numbers, letters, or underscore).
It also ignores automatically all declined and tentative-attendance ("maybe") events.
